```
🔍 Tab Search - 跨窗口 Tab 搜索

厌倦了在几十个 Tab 中翻找？Tab Search 让你在所有 Chrome 窗口中快速定位并跳转到指定 Tab。

✨ 核心功能
• 跨窗口搜索：搜索所有打开的 Chrome 窗口，不只是当前窗口
• 实时过滤：输入即搜，支持标题和 URL 模糊匹配
• 一键跳转：选中后自动聚焦窗口并激活 Tab
• 键盘导航：全程键盘操作，无需鼠标
• 智能分组：按窗口分组显示，当前窗口优先

⌨️ 快捷键
• Ctrl+Shift+T（Mac: ⌘+Shift+T）：打开搜索面板
• ↑/↓：上下选择结果
• Enter：跳转到选中的 Tab
• Esc：关闭搜索面板

🎯 使用场景
• 打开了多个窗口，每个窗口有大量 Tab
• 需要快速找到之前打开过的页面
• 只记得页面标题关键词，不记得在哪个窗口

🔒 隐私说明
本插件仅在本地运行，不会收集或上传任何数据。需要的权限仅用于读取 Tab 信息和激活 Tab。

💡 提示
安装后建议将插件固定到工具栏，方便随时使用。
```

## 🚀 发布步骤

### 第一步：注册开发者账号

1. 访问 [Chrome Web Store 开发者控制台](https://chrome.google.com/webstore/devconsole/)
2. 登录 Google 账号
3. 支付 **$5** 一次性注册费（信用卡或 PayPal）
4. 填写开发者信息

### 第二步：上传扩展

1. 点击「**新建项目**」→「**添加新的扩展程序**」
2. 上传 `chrome-tab-search-store.zip`
3. 填写商店信息（使用上面的模板）
4. 上传宣传图片（截图等）

### 第三步：设置价格/分发

- **价格**：免费（Free）
- **可见性**：公开（Public）
- **地区**：全球（All regions）

### 第四步：提交审核

1. 点击「**提交审核**」
2. 等待审核（通常 1-3 个工作日）
3. 审核通过后自动上架

---

## ⚠️ 审核注意事项

### 权限说明（重要）

manifest.json 中的权限需要在描述中解释：
- `tabs`：用于读取所有窗口的 Tab 信息
- `activeTab`：用于激活选中的 Tab

### 避免审核被拒绝

✅ **应该做：**
- 准确描述权限用途
- 提供清晰的隐私政策链接（如需要）
- 确保代码没有恶意行为
- 截图真实反映功能

❌ **不要做：**
- 请求不必要的权限
- 包含混淆代码（除非必要且说明）
- 使用误导性描述
- 包含远程加载的代码

## 📋 发布后维护

### 更新版本

1. 修改 `manifest.json` 中的 `version`（如 "1.0.0" → "1.0.1"）
2. 重新打包 zip
3. 在开发者控制台上传新版本
4. 提交审核

在开发者控制台可查看：
- 安装量
- 活跃用户
