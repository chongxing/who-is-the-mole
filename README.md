# 🕵️‍♂️ 谁是卧底

> **本游戏代码完全由 OpenClaw AI 助手生成**

一个用于线下聚会的文字推理游戏助手，帮助主持人快速分发词语、管理游戏流程。

## 🎮 游戏简介

"谁是卧底"是一款流行的派对游戏：
- **平民**：N-1 人拿到相同的词语
- **卧底**：1 人拿到相近但不同的词语
- 玩家根据词语描述进行推理，找出卧底

## ✨ 功能特点

- 🎯 **隐私保护**：依次显示词语，防止偷看
- 🎭 **身份隐藏**：玩家只知道词语，不知道自己是平民还是卧底
- 💾 **记忆功能**：自动保存上次选择的人数
- 📱 **过渡动画**：精美的手机传递页面
- 🎨 **现代 UI**：深色主题 + 玻璃拟态效果
- 📊 **主持人面板**：游戏结束后显示完整身份揭秘

## 📖 使用说明

1. **设置人数**：选择 3-20 名玩家
2. **开始游戏**：点击"开始游戏"按钮
3. **查看词语**：每位玩家依次查看自己的词语
4. **传递手机**：点击"我记住了，下一个"
5. **开始推理**：所有玩家查看完毕后，开始描述词语
6. **投票环节**：每轮投票选出最可疑的人
7. **查看结果**：游戏结束后查看完整身份揭秘
